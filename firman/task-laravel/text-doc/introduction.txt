Soal :

Jawaban :
1. Laravel adalah framework PHP open source yang digunakan untuk membangun aplikasi web. Laravel dirancang untuk membuat pengembangan aplikasi web yang lebih efisien, dinamis dan powerfull
2. Beberapa fitur yang ada pada Laravel diantaranya, template yang ringan, arsitektur MVC, Tool Artisan, Eloquent ORM, Authentication,  Data Migration
